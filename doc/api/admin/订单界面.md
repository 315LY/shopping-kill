### 订单界面内容

#### 根据订单类型获取内容
方式：GET

地址：/admin/getOrderAllById/v1?type=4

参数：

|参数|类型|是否必须|说明|
|---|---|---|---|
|type|Integer|否|0-取消订单 1-未支付 2-已支付 3-已出库 4-已收货 5-已评价|
|id|Long|否|订单id|
|userId|Long|否|用户id|
|goodsId|Long|否|商品id|
|name|String|否|商品名|
|nickName|String|否|用户名|
|PS：type与其他属性必须存在一个|
||
|current|Long|否|默认1|
|size|Long|否|默认10|
