#### 后端首页顶部展示
方式: GET

地址：/admin/getTotalToday/v1

参数：无

返回值：
```json
{
    "code": 0,
    "msg": "SUCCESS",
    "userMsg": "操作成功",
    "data": {
        "Number": 9,
        "TodayOrder": 0,
        "TodayNum": 0,
        "TodayOut": 0
    }
}
```
